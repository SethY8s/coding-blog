This is a simple blog site that allows users to submit a form that is posted to the node.js backend and stored in a local JSON file. 
The website on load, will parse the JSON convert the data to html for people to see.

TO RUN:
-in the terminal type "npm run dev"
-then go to local host 5000 on the browser
-Type in a blog and post it. On refresh your data should remain on the page.(currently there is no delete button)
